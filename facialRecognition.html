Facial Recognition in Python

I coded a headless Raspberry Pi and camera module in Python to 1) detect and save images of a user's face to a directory with their name, 2) train a machine learning model on the images of the desired user's face versus the faces of other users, and 3) use the machine learning model to recognize the desired user's face among other faces.

To detect the user's face, I used Open Source Computer Vision, or OpenCV, and Object Detection using Haar feature-based cascade classifiers.

Haar Features.

Object Detection using Haar feature-based cascade classifiers uses edge features, line features, and four-rectangle features. The algorithm applies the features for each possible size and location and computes a single value by subtracting the sum of pixels under the white area of the feature from the sum of pixels under the black area of the feature.

Adaboost selects the best features from all possible features computed by selecting the features with the lowest error rate. The algorithm computes each feature for all training images and weights the feature by analyzing the errors it produces and the weighting of the image it's being applied to, as images more frequently misclassified are weighted higher.

Cascade of Classifiers is then used, in which features are grouped into stages of classifiers and they are applied one-by-one. If an image fails a stage, it is discarded before future stages are applied. If a window on the image passes all stages, it is recognized as a face.

First, the system asks for the user's name and creates a directory for them. It then captures 30 images. For each image, the system applies Haar features to detect a face. If a face is detected, the window of the image with the face is saved to that user's directory.

A face is detected and saved to the Courtney directory.

Second, the system is trained on the images of each person in their respective directory.

Third, the system continuously captures images. For each image, the system applies Haar features to detect a face. If a face is detected, the machine learning algorithm compares the window of the image with the face against the training data. If the confidence, or distance to the closest item in the training data, is low enough, the system prints out the name of the user it recognizes and its confidence!

Courtney's dad's face is not recognized.

Courtney's face is recognized with confidence 67.5!
