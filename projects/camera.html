Image Stitching

Class Project

Harris Corner Detector

I performed corner detection with the Harris Corner Detector. I first computed image gradients with the Sobel filter. Then, I computing the covariance matrix of the image gradients convolved with the average filter. Then, I computed the Harris response for each pixel. Finally, I applied non-maximal suppression by removing responses that are not local maxima.
