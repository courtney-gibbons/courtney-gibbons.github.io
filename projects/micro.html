Programming Microcontrollers Projects

My programming microcontrollers projects include two 1st place and one 4th place hardware hackathon wins! I also TA Intro to ECE at UCLA teaching freshmen how to program a path following car.

Class Project & TA Position

Path Following Car

I coded a car in C that would follow a dark path on a light background by reading in IR sensor values, weighting the sensor values by their distance to the center of the car, computing the error, and implementing a PID controller to change the speed of the car's individual wheels in response to the error. The professor invited me back to TA the class, and I ran lab sections for 60 students.

Winning Hackathon Project

Zoomsday Clock

Zoomsday Clock is a warning indicator for video conferencing. It displays a red light if the user is in a meeting and counts down how much time is left. A wave of the hand reveals the time when that meeting ends. It won 1st place in the 2021 UCLA IDEA Hacks, the largest hardware hackathon on the West Coast. It includes a web control panel and IR remote control. We used C++ code on the WiFi-connected ESP32.
